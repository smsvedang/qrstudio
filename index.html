4. Line 11 pe APPS_SCRIPT_URL mein Step 2 ka URL daalo

5. Save and Deploy → Worker URL copy karo

User Guide
Complete setup aur usage guide
◈ QR Studio kya hai?

12 types ke QR codes banao, logo lagao, bulk generate karo, aur Cloudflare Worker + Google Sheets se scan tracking karo. Koi server kharidna nahi, koi cost nahi.

A
Setup (ek baar)

Step 1 — Google Sheet: sheets.new → nayi sheet → URL se ID copy karo

Step 2 — Apps Script: script.google.com → New project → Code.gs tab se code copy → paste → Sheet ID daalo → Deploy as Web app (Me/Anyone) → URL copy karo

Step 3 — Cloudflare Worker: cloudflare.com → Workers → Create → Edit Code → worker.js tab se code copy → paste → Line 11 pe Apps Script URL daalo → Deploy → Worker URL copy karo

Step 4: Setup tab → Worker URL paste karo → Test karo ✓

B
QR Code Banana

Create QR → type select karo → details bharo → style/size/logo choose karo → Generate → PNG/SVG download karo → Print!

C
⭐ URL Badalna Bina Reprint Kiye

Tracked QRs tab → kisi bhi QR ke saamne ✎ Edit → nayi URL daalo. Same printed QR, naya destination. Poster, sticker, parcel — kuch bhi reprint nahi karni!

Common Problems

❓ Redirect nahi ho raha: Browser mein Worker URL/health kholo — JSON aana chahiye.

❓ Sheets mein data nahi aa raha: Worker mein APPS_SCRIPT_URL check karo. Apps Script Access: "Anyone" hona chahiye.

❓ QR not found: Worker URL Setup tab mein sahi hai? QR save hua tha? Tracked QRs tab mein dekho.
