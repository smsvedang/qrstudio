User Guide
QR Studio ka complete setup aur usage guide
◈ QR Studio kya hai?

Yeh ek complete QR code tool hai — 12 types ke QR banao, logo lagao, bulk generate karo, aur URL tracking ke liye Google Sheets mein scan data log karo. Koi server nahi, koi cost nahi.

A
Pehli Baar Setup (Tracking ke liye)

1. sheets.new pe nayi Google Sheet banao

2. URL se Sheet ID copy karo:

https://docs.google.com/spreadsheets/d/YAHAN_WALA_ID_COPY_KARO/edit

3. ⌨ Code.gs sidebar tab pe jao → saara script code copy karo

4. script.google.com → New project → Code.gs mein paste karo

5. Line 2 pe SHEET_ID mein apna Sheet ID daalo → Save

6. Deploy → New deployment → Type: Web app → Execute as: Me → Access: Anyone → Deploy

7. URL copy karo → ⚙ Setup tab mein paste karo

B
QR Code Banana

1. ✦ Create QR tab pe jao

2. Type select karo — URL, WiFi, UPI, Contact, etc.

3. Details bharo → Style aur size choose karo

4. Optionally logo upload karo (neeche waala section)

5. ✦ Generate QR Code dabao → PNG/SVG/Hi-Res download karo

C
Logo Overlay — QR pe apna logo lagao

Style section mein Logo / Image Overlay mein PNG/JPG upload karo.

Position choose karo — center sabse common hai branded QRs ke liye.

Error Correction ko H (High) pe rakhna recommended hai jab logo center mein ho.

Download karo → Logo automatically QR ke upar composite ho jaata hai.

D
⭐ URL Badalna Bina Reprint Kiye

URL type ke tracked QRs mein — Tracked QRs tab → ✎ Edit → nayi URL daalo.

Same printed QR, new destination. Posters, stickers, parcels — kuch bhi print ho — URL baad mein change ho sakti hai.

Common Problems

❓ Scan hota hai lekin Sheets mein data nahi: Apps Script → Manage deployments → Access: "Anyone" confirm karo. Naya deployment banana pad sakta hai.

❓ Test connection fail: Normal hai — CORS browser mein block karta hai. Seedha phone se QR scan karke test karo.

❓ Location data blank: Local/VPN IP se geo kaam nahi karta. Public IP se scan karo.
