1 Google Sheet banao

Nayi Google Sheet banao, URL se Sheet ID copy karo:

https://docs.google.com/spreadsheets/d/YAHAN_SE_COPY_KARO/edit

Sheet mein 2 tabs automatically ban jayenge: QR_Codes aur Scans

2 Apps Script deploy karo

1. script.google.com kholo
2. New Project → Code.gs mein saara code paste karo (jo file diya hai)
3. SHEET_ID wali line mein apna Sheet ID daalo
4. Deploy → New deployment → Web app
5. Execute as: Me | Access: Anyone
6. Deploy karo → URL copy karo

3 URL yahan paste karo
4 Done! ✓

Ab QR banao → koi bhi scan kare → data Google Sheets mein automatically log hoga.

Tracking URL format: https://script.google.com/macros/s/XXXXX/exec?id=your-id
